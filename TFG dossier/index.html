Arquitectura de sistemes big data pel record linkage de xarxes socials


En aquest index es poden veure les diferents seccions del dossier del TFG. Els hipervincles redireccionen a l'apartat del PDF que es seleccionat. Les primeres versions dels informes, que es poden visualitzar en l'apartat del index "5. Versions inicials dels informes", contenen anotacions per tal de veure més fàcilment els canvis realitzats i tenen subratllat les parts del texte modificades. També hi ha un apartat amb una llista de tots els canvis afegits degut als comentaris del tutor. Es recomana l'utilització del navegador web Firefox com a primera opció ja que les anotacions es visualitzen millor, o bé Chrome com a segona opció.
